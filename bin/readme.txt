To execute a file run the himlcompiler.exe with one argument which points to the iml file.
Note: You need to write "\\" instead of "\" for a Path in the Windows environment

himlcompiler.exe <AbsolutePath>

Example:
    himlcompiler.exe "C:\\cpib\\CPIB_ILMCompiler\\showcases\\3np1.iml"
or on Windows 10 use "/"
    himlcompiler.exe "C:/cpib/CPIB_ILMCompiler/showcases/3np1.iml"
or on macOS use
    ./mac_himlcompiler "3np1.iml"
